# PetMap
**PetMap** — це соціальна платформа для допомоги тваринам і координації взаємодії між небайдужими людьми, волонтерами та притулками.

Користувачі можуть додавати на мапу інформацію про **знайдених, загублених або тварин, які потребують допомоги**, вказуючи їхній вид, стан, приблизне місцезнаходження та додаткові деталі, зокрема чи є тварина потенційно домашньою та її кличку (за наявності).

Інші учасники спільноти можуть відреагувати: забрати тварину, доставити до притулку або надати тимчасову допомогу. **PetMap показує статус кожного випадку, що запобігає дублюванню дій і хаосу.**
